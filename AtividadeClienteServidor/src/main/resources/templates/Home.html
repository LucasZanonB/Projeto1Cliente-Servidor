Bem-vindo! Esta é a rota raiz.

Nesse projeto temos essa rota principal que descreve o projeto. Alem da rota principal temos mais 3 rotas,uma para calcular o IMC,outra para retornar qualquer texto inserido pelo usuario e a ultima que o usuario insere duas notas e elas são somadas.Nessa apresentação das rotas,onde tem as chaves, devemos inserir os valores sem elas. As mesmas são indicaçao que precisa de uma substituição pelos valores

Rota principal: http://localhost:3500/

http://localhost:3500/imc/{peso}/{altura}

http://localhost:3500/texto/{texto}

http://localhost:3500/nota/{nota1}/{nota2}
